## About This Template

This project is built with **buildbook**, a Pandoc-based book build system.
Your manuscript lives as plain Markdown files in the `content/` directory.
Running a single command compiles all chapters into a polished Word document,
EPUB, PDF, or HTML file.

Key ideas behind the workflow:

- **Write in Markdown** — focus on words, not formatting.
- **Version in Git** — every draft is tracked, every change is reversible.
- **Build with Pandoc** — one source, many output formats.

## How to Get Started

1. Replace the files in `content/` with your own chapters.
2. Update `content/book.yaml` with your title, author, and version.
3. Run the build:

```bash
buildbook build -f docx -o my-book
```

## Structure of This Book
